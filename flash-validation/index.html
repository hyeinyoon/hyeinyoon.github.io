Notes:
- This tool uses ASKAPsoft outputs. FITS-datacubes are needed only for getting major and minor beam sizes (from the header).
- The values and plots in the tables below are not currently correct - I used dummy files for some cases to run the script successfully.
- 1) Total bandwidth: 288-MHz (FLASH) vs 12-MHz (WALLABY) - Should we split the long bandwidth into smaller units for validation check?
- 2) Embedded HI absorption/emission lines? How many lines are expected in the field? Are they negligible in flux/rms measurements of each beam?
- 3) Validation Checking with extracted spectral line domain? With well-known strong radio sources in the field?
- 4) Any other additional items to be required (e.g. FIRST detections in FOV)?
- 5) Goal of the tool: The tool does not make judgements (i.e. the data quality is good or bad). Do we aim to provide any metrics or quantify the data quality?
